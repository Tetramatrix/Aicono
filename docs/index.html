🤖 What is Aicono?

Aicono is a Windows-only freeware app that uses the latest AI technology to automatically group desktop icons by category (e.g., "Work," "Games," "Utilities") and to arrange them neatly without overlaps in a customizable desktop area. Fine-tune with a visual preview tool! 🧠 No more manual dragging or cluttered screens.

⭐ Key Features

  • 🧠 AI-Powered Grouping: Automatically categorizes icons logically into groups like Work, Games, and Utilities.
  • 🎛️ Your AI, Your Choice: Works 100% offline with a lightweight default model (~806MB) that downloads on first run. Want more power? Aicono is built for enthusiasts:
    • Connect to local services like Ollama or any OpenAI-compatible server.
    • Configure it to auto-download larger, more powerful models (e.g., 8B models) from Hugging Face.
    • Supports private models using your Hugging Face token.
    • Includes a manual fallback if no AI model is available.
  • 💾 Instant Fine-Tuning with Caching: Don't wait for the AI analysis twice! Aicono saves your layout. If you're unhappy with the arrangement, just re-open the app to instantly make adjustments (like changing spacing or grid size) and re-apply in seconds. The "Clear Cache & Reload" button is only for when you want to start over.
  • 📐 Intelligent Layout: A sophisticated spatial algorithm arranges icons neatly, ensuring no overlaps.
  • 📍 Customizable Area: Select specific desktop regions for icon placement.
  • 👁️ Visual Preview (WYSIWYG): See and approve the new layout before any changes are applied.
  • ⚙️ Fully Configurable: All settings, from AI models to layout, are controlled via a simple JSON file for power users.
  • 📦 Portable App: No installation required – just run and go!
  • 💰 Freeware: Free community edition.

⚙️ How Does It Work?

🧠 Fully automatic & user-friendly! :

  1. 👁️ Scans desktop icons in real-time.
  2. 🧠 Uses AI to categorize (Work, Games, etc.).
  3. 📐 Applies a spatial layout without overlaps.
  4. 👀 Offers a Preview for fine-tuning.
  5. 🔧 Customizes to your needs with area selection.
  6. 💾 Caches results for instant re-application.

🧠 A Note on the AI: "Why are my results 'hit-or-miss'?"

This is an intentional feature! Aicono is built for both casual users and AI enthusiasts.

  • Offline & Free (Default): On its first run, Aicono downloads a lightweight, built-in model (~806MB). This allows it to work completely offline for free. As a smaller model, it's fast but may occasionally mislabel complex icons as "Miscellaneous."
  • Power-User Ready (Custom): If you're an AI enthusiast and want perfect results, you can easily boost the app's "brain." Aicono is designed to connect to:
    • Local services like Ollama or Llama.
    • Any OpenAI-compatible server.
    • Larger, more powerful models (like 8B parameter models) that you can configure Aicono to auto-download from Hugging Face.

💡 Pro-Tips for a Perfect Layout 💡

Aicono is a powerful tool with many controls. Like any good tool, it's designed to give you the power to find your own perfect, individual result. Here are two key tips to get started.

1. ⚠️ REQUIRED: Disable Windows "Auto-Arrange"

Aicono needs full control to move your icons. If Windows is also trying to manage them, your layout will break or won't apply correctly.

To check this: Right-click your Desktop > go to "View" > and make sure "Auto-arrange icons" is UNCHECKED. ✅

2. 📐 Balancing Your Working Area and Layout Controls

Think of the "Working Area" as a "sandbox" 🖼️ or a "fence" to contain your icons. Its only job is to ensure icons don't go outside the boundaries you set.

All the other controls 🎛️—like "Grid Density", "Gaps", and "Spacing"—determine how your icon groups will fit inside that sandbox.

There is no single "right" setting! The best layout is a balance between the size of your sandbox and the layout controls you choose. If things look cramped 😖, try giving them more room (a larger area ↔️) or tightening up the spacing (a smaller gap ↕️).

We encourage you to experiment! 🎨 That's what the instant WYSIWYG (What You See Is What You Get) preview is for. Adjust any slider and watch your layout change in real-time.
