Why are my results "hit-or-miss"?

Aicono uses AI models to group and arrange icons. The default model is lightweight and works offline, but may not always classify every icon perfectly, especially for less common applications. Advanced users can connect larger or custom models for improved accuracy, but this may require more system resources.

  • Offline & Free (Default): The built-in model (~806MB) is downloaded on first run and works fully offline. It is fast, but may sometimes mislabel complex icons as "Miscellaneous."
  • Custom Models: For improved results, Aicono can connect to local services (like Ollama or Llama), OpenAI-compatible servers, or download larger models from Hugging Face.

Note: Larger models can provide better grouping, but require more memory and processing power.

Manual Fine-Tuning & Fallback

If the AI does not classify an icon as desired, or if no model is available, you can adjust the results manually:

  1. Fine-tune AI results: After each run, Aicono saves the grouping in icon_groups_cache.json. You can edit this file to move icons between groups. Your changes will be used until you clear the cache.
  2. Manual fallback: If no AI is available, you can create icon_groups_cache.json from scratch and define all groups manually. The app will use your structure.

File Structure (icon_groups_cache.json)

The file must be saved as JSON with the following structure:
